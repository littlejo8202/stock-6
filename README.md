# 전통/혁신 테마 별 포트폴리오 성과 비교 - 데이터처리프로그래밍 6조

### 1. 기본 설명
1. 포트폴리오 구성 비율: 기본구성(금, 채권) 30% / 테마종목 70%
2. 테마종목 구성: 세부 종목은 산업별 대표 top5를 사용했으며, 국내/해외 테마별 etf를 사용하였습니다.
3. 기본종목 구성: 금과 채권으로 구성하였습니다.

### 2. 폴더 구조
   - 'backend' - data (사용한 데이터들), app.py (백엔드 코드)
   - 'front/src' - data (설문 내용), pages (프론트엔드 코드), styles (css)
